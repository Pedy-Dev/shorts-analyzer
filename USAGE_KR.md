## 🔑 API 키 설정

오른쪽 상단의 **⚙️ API 키 설정** 버튼을 클릭하여 필요한 API 키를 입력하세요.

### 필수 API 키

#### 1. YouTube Data API 키
- **용도**: YouTube 영상 정보 및 자막 수집
- **발급 방법**:
  1. [Google Cloud Console](https://console.cloud.google.com/) 접속
  2. 새 프로젝트 생성
  3. "API 및 서비스" > "라이브러리" 이동
  4. "YouTube Data API v3" 검색 후 사용 설정
  5. "사용자 인증 정보" > "API 키 만들기"

#### 2. Google Gemini API 키
- **용도**: AI 대본 구조 분석 및 지침 생성
- **발급 방법**:
  1. [Google AI Studio](https://makersuite.google.com/app/apikey) 접속
  2. "Get API key" 클릭
  3. API 키 생성 및 복사

#### 3. Google OAuth 클라이언트 (선택사항)
- **용도**: 내 채널 분석 (YouTube Analytics 데이터 조회)
- **발급 방법**:
  1. [Google Cloud Console](https://console.cloud.google.com/) 접속
  2. "API 및 서비스" > "사용자 인증 정보"
  3. "YouTube Analytics API" 사용 설정
  4. OAuth 동의 화면 구성
  5. OAuth 2.0 클라이언트 ID 생성

## 📊 타 채널 분석

다른 채널의 Shorts 영상을 분석하여 성공 패턴을 파악합니다.

### 사용 방법

#### 1단계: 채널 URL 입력
```
예시: https://www.youtube.com/@channelname
```

#### 2단계: 분석할 영상 개수 선택
- 10개 / 20개 / 30개 / 50개 중 선택
- 많을수록 정확하지만, 시간이 오래 걸립니다

#### 3단계: 분석하기 버튼 클릭
- 자동으로 최근 Shorts 영상 수집
- 각 영상의 자막 자동 추출
- 조회수, 좋아요, 댓글 등 통계 수집

#### 4단계: 대본 구조 분석
- **"1단계: 대본 구조 분석하기"** 버튼 클릭
- AI가 자막을 분석하여:
  - 채널 DNA (영상 구조, 말투, 스타일)
  - 성과 좋은 영상의 특징
  - 개선이 필요한 부분

#### 5단계: 대본 제작 지침 생성
- **"2단계: 대본 제작 지침 생성하기"** 버튼 클릭
- AI가 분석 결과를 바탕으로 맞춤 대본 제작 지침 생성
- **📋 전체 복사** 버튼으로 클립보드에 복사

### 분석 결과 해석

#### 📊 분석 결과
- **처리 영상**: 수집된 전체 영상 개수
- **성공**: 자막 수집 성공한 영상
- **실패**: 자막이 없거나 추출 실패
- **평균 조회수/좋아요/댓글**: 채널의 평균 성과

#### 🧬 채널 DNA
- **영상 구조**: 도입-전개-반전-결말 비율
- **문장 길이**: 짧음/보통/김
- **말투**: 반말/존댓말/유머 스타일 등
- **콘텐츠 유형**: 정보형/엔터테인먼트/교육 등

#### 🔥 성과 좋은 영상의 비밀
- 조회수/참여도가 높은 영상의 공통점
- 구체적인 예시와 함께 제공

#### ⚠️ 개선 필요한 부분
- 성과가 낮은 영상의 문제점
- 개선 방안 제시

## 🎯 내 채널 분석

내 YouTube 채널의 Shorts 성과를 상세히 분석합니다.

### 사용 방법

#### 1단계: Google 로그인
- **"Google 계정으로 로그인"** 버튼 클릭
- YouTube 채널과 연결된 Google 계정으로 로그인
- YouTube Analytics 데이터 접근 권한 허용

#### 2단계: 영상 분석 시작
- 분석할 영상 개수 선택 (10~50개)
- **"영상 분석 시작"** 버튼 클릭

#### 3단계: 분석 결과 확인
아래 지표들을 확인할 수 있습니다:
- **조회수**: 총 조회수
- **유효조회수**: 실제 시청한 조회수
- **좋아요/댓글/공유**: 참여도 지표
- **평균 시청시간**: 영상을 얼마나 오래 봤는지
- **평균 조회율**: 영상 길이 대비 시청률
- **구독자 증가**: 해당 영상으로 인한 구독자 변화

#### 4단계: 대본 보기
- 각 영상 옆의 **"📝 대본 보기"** 버튼 클릭
- 자막 전체 내용 확인 가능

### 채널 전환
- 다른 채널로 분석하려면 **"채널 전환"** 버튼 클릭
- 다시 로그인하여 다른 채널 선택

## 💡 활용 팁

### 1. 벤치마킹 전략
- 같은 분야의 성공한 채널을 3~5개 분석
- 공통 패턴 찾기
- 내 채널에 적용할 요소 선택

### 2. 대본 제작 워크플로우
1. 타 채널 분석으로 성공 패턴 파악
2. 생성된 지침을 ChatGPT/Claude에 입력
3. AI에게 "이 지침에 따라 대본 작성해줘" 요청
4. 검토 후 영상 제작

### 3. 내 채널 개선
1. 내 채널 분석으로 현재 성과 파악
2. 성과 좋은 영상 vs 낮은 영상 비교
3. 개선점 파악 후 다음 영상에 적용

### 4. A/B 테스팅
- 분석 지침을 바탕으로 2가지 스타일 테스트
- 어떤 스타일이 내 채널에 맞는지 실험

## ⚠️ 주의사항

1. **API 할당량 제한**
   - YouTube Data API는 일일 할당량 제한이 있습니다
   - 너무 많은 영상을 분석하면 할당량 초과 가능

2. **자막 없는 영상**
   - 자막이 없는 영상은 분석할 수 없습니다
   - 자막 자동 생성 기능을 켜둔 채널만 분석 가능

3. **분석 시간**
   - 영상 개수에 따라 5~10분 소요될 수 있습니다
   - 자막 수집이 가장 시간이 오래 걸립니다

4. **API 키 보안**
   - API 키는 브라우저의 localStorage에 저장됩니다
   - 공용 컴퓨터에서는 사용 후 API 키를 삭제하세요

## 🐛 문제 해결

### "유효한 채널 URL이 아닙니다"
- YouTube 채널 URL 형식 확인
- 예시: `https://www.youtube.com/@channelname` 또는 `https://www.youtube.com/channel/UC...`

### "Shorts 영상을 찾을 수 없습니다"
- 해당 채널에 Shorts가 없거나
- API 키가 올바르지 않을 수 있음

### "자막 추출 실패"
- 영상에 자막이 없거나
- 자막이 비공개로 설정된 경우

### "구조 분석 실패"
- Gemini API 키 확인
- 할당량 초과 여부 확인

### "로그인 실패"
- OAuth 클라이언트 ID/Secret 확인
- Redirect URI가 올바른지 확인 (`http://localhost:3000/api/auth/callback`)
